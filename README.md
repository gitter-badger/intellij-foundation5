Intellij Zurb Foundation 5 plugin
==================================

An IntelliJ plugin containing Zurb foundation 5 live templates! This plugin works for the following JetBrains products:

- IntelliJ IDEA
- WebStorm and PhpStorm
- PyCharm
- RubyMine
